# Password_Bruteforce
This program attempts to guess a password by iterating through a list of possible passwords stored in a text file. This is an attempt to simulate how attackers try to gain unauthorized access when weak passwords are used.
# USE FOR EDUCATIONAL PURPOSES ONLY!
---
## Features
- Reads passwords from a text file.
- Attempts each password one by one.
- Displays each attempt in the terminal
- Stops when the correct password is found.
---

Place the wordlist file with the script in the same folder.
